# Mark and Justin's Raytracer (MJR)
Created from scratch as a class project for CS 184

## Build Instructions
From the directory, simply run the following commands:
~~~~
mkdir build
cd build
cmake ..
make
./mjr
~~~~
## Running the Test Suite
Almost the same as above:
~~~~
mkdir build
cd build
cmake -Dtest=ON ..
make
./test/unitTests
~~~~
